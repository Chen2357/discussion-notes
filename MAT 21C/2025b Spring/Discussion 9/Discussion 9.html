§I Warm-Up

(Warm-up) Identify where along the trail (shown in blue) is the hiker’s elevation the greatest (the level curves are shown in gray).

Solution. The hiker is at the greatest elevation at the red point in the following figure. The hiker reaches the elevation of around \(220\) as seen by the level curve.

One way to describe the point is that it is where the trail is tangent to a level curve. After all, if the trail is crossing a level curve, then we are either going uphill or downhill, so we are not at the greatest elevation. Mathematically, such a point can be found by finding the gradient of the elevation function and solve for when the gradient is perpendicular to the velocity vector of the trail using the dot product. This works because the gradient is always perpendicular to the level curve, so if the velocity vector is perpendicular to the gradient, then it must be tangent to the level curve.

§II Gradient

Recall that the difference \(f(x + dx, y + dy) - f(x, y)\) for small \(dx\) and \(dy\) can be approximated by $$\frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dy = \left\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right\rangle \cdot \langle dx, dy \rangle.$$ We call the first vector in the dot product, the gradient of \(f\), and we denote it by $$\nabla f = \left\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right\rangle.$$ Besides using it for approximation, it has two other geometric meanings.

  1. The gradient points in the direction of the steepest increase, meaning that the directional derivative $$D_{\vec{u}} f = \nabla f \cdot \vec{u}$$ is the greatest among unit vectors \(u\) precisely when \(u\) points in the same direction as the gradient. The magnitude of the gradient is precisely the value of the greatest directional derivative.
  2. The gradient of \(f(x, y)\) is perpendicular to the level curves, and so it is a normal vector to the tangent line of the level curve. In particular, the tangent line to the level curve of \(f(x, y)\) at \((x_0, y_0)\) is given by $$\nabla f (x_0, y_0) \cdot \langle x - x_0, y - y_0 \rangle = 0.$$ In \(3\)-dimensions, the tangent plane to the level surface of \(f(x, y, z)\) at \((x_0, y_0, z_0)\) is given by $$\nabla f (x_0, y_0, z_0) \cdot \langle x - x_0, y - y_0, z - z_0 \rangle = 0.$$

(Exercise)

  1. Find the gradient of \(f(x, y) = x^2 - xy + \frac{1}{2} y^2 + 3\). Then, evaluate \(\nabla f (3, 2)\).
  2. Verify that \((3, 2, 8)\) is on the surface \(z = f(x, y)\) and find the equation for the tangent plane of the surface at that point. Hint: the surface can also be written as \(f(x, y) - z = 0\).
  1. Approximate the value of \(f(3.01, 2.1)\), call it \(L(3.01, 2.1)\).
  2. The approximation you obtained in the previous part is called the linear approximation. The error \(E(3.01, 2.1) = f(3.01, 2.1) - L(3.01, 2.1)\) is bounded by $$\lvert E(3.01, 2.1) \rvert \leq \frac{1}{2} M (\lvert 3.01 - 3 \rvert + \lvert 2.1 - 2 \rvert)^2$$ where \(M\) must be an upper bound of \(\lvert f_{xx} \rvert\), \(\lvert f_{yy} \rvert\), and \(\lvert f_{xy} \rvert\) for all points \(3 \leq x \leq 3.01\) and \(2 \leq y \leq 2.1\). Find the error bound.

Solution.

  1. The gradient of \(f(x, y)\) is $$\nabla f = \left\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right\rangle = \langle 2x - y, -x + y \rangle.$$ Evaluating at \((3, 2)\), we get $$\nabla f (3, 2) = \langle 4, -1 \rangle.$$
  2. The point \((3, 2, 8)\) is on the surface \(z = f(x, y)\) because $$f(3, 2) = 3^2 - 3 \cdot 2 + \frac{1}{2} \cdot 2^2 + 3 = 8.$$ We can find the tangent plane by finding the gradient of \(f(x, y) - z\) which is $$\nabla (f(x, y) - z) = \langle 2x - y, -x + y, -1 \rangle.$$ Evaluating at \((3, 2, 8)\), we get the normal vector to the tangent plane $$\langle 4, -1, -1 \rangle.$$ Thus, the tangent plane is the plane perpendicular to \(\langle 4, -1, -1 \rangle\) that passes through \((3, 2, 8)\), which is given by $$\langle 4, -1, -1 \rangle \cdot \langle x - 3, y - 2, z - 8 \rangle = 0.$$
  3. We can approximate the difference \(f(3.01, 2.1) - f(3, 2)\) by the linear approximation $$\nabla f (3, 2) \cdot \langle 0.01, 0.1 \rangle = \langle 4, -1 \rangle \cdot \langle 0.01, 0.1 \rangle = -0.06.$$ Thus, we can approximate \(f(3.01, 2.1)\) by $$L(3.01, 2.1) = f(3, 2) - 0.06 = 8 - 0.06 = 7.94.$$

  4. The second derivatives of \(f\) are $$f_{xx} = 2, \quad f_{yy} = 1, \quad f_{xy} = -1.$$ Thus, we can take \(M = 2\) as an upper bound for the second derivatives, and the error is bounded by $$\lvert E(3.01, 2.1) \rvert \leq \frac{1}{2} \cdot 2 \cdot (0.01 + 0.1)^2 = 0.0121.$$ This tells us that the actual value of \(f(3.01, 2.1)\) is between \(7.94 - 0.0121\) and \(7.94 + 0.0121\).

    If you are curious, the actual value of \(f(3.01, 2.1)\) is exactly \(7.9441\). Our approximation is pretty close and our error bound is correct.

In general, the linear approximation of a function \(f(x, y)\) at a point \((x_0, y_0)\) is given by $$L(x, y) = f(x_0, y_0) + \nabla f (x_0, y_0) \cdot \langle x - x_0, y - y_0 \rangle.$$ The error of the linear appximation on a rectangle containing \((x_0, y_0)\) is bounded by $$\lvert E(x, y) \rvert \leq \frac{1}{2} M (\lvert x - x_0 \rvert + \lvert y - y_0 \rvert)^2$$ where \(M\) is an upper bound of the second derivatives of \(f\) for all points in the rectangle.

§III Second Derivative Test

If we want to find the extrema of a function \(f(x)\) of one variable, we can find the critical points by solving \(f'(x) = 0\), and then determine which critical points are local maxima or local minima by using the second derivative test. To recap, the second derivative test for one variable says that:

  • If \(f''(x) > 0\), then the function is “happy” (curving upward), so the critical point is a local minimum.
  • If \(f''(x) < 0\), then the function is “sad” (curving downward), so the critical point is a local maximum.

For a function \(f(x, y)\) of two variables, we can find the critical points by solving $$f_x = 0 \quad \text{and} \quad f_y = 0.$$ At critical points, the function is flat like the one variable case. A naive classification of the critical points might be to just look at the second derivatives \(f_{xx}\) and \(f_{yy}\):

  • A local maximum is curving downward in both directions, so we expect \(f_{xx} < 0\) and \(f_{yy} < 0\).
  • A local minimum is curving upward in both directions, so we expect \(f_{xx} > 0\) and \(f_{yy} > 0\).
  • If \(f_{xx}\) and \(f_{yy}\) have different signs, then the function is curving upward in one direction and curving downward in the other, so the critical point is a saddle point.

This characterization is almost complete. The fact that a surface is curving upward in both axes directions does not mean that it is curving upward in all directions: if it dips along the diagonal, then it is a sadle after all.

To take the diagonal effect into account, the complete second derivative test for two variables look at the so-called discriminant \(f_{xx} f_{yy} - f_{xy}^2\).

  • If \(f_{xx} f_{yy} - f_{xy}^2 > 0\) and \(f_{xx} > 0\) (or \(f_{yy} > 0\)), then the critical point is a local minimum.
  • If \(f_{xx} f_{yy} - f_{xy}^2 > 0\) and \(f_{xx} < 0\) (or \(f_{yy} < 0\)), then the critical point is a local maximum.
  • If \(f_{xx} f_{yy} - f_{xy}^2 < 0\), then the critical point is a saddle point.
  • If \(f_{xx} f_{yy} - f_{xy}^2 = 0\), the test is inconclusive.

Note that our naive classification is a special case of the discriminant test.

  • For local extrema, we have \(f_{xx} f_{yy} - f_{x y}^2 > 0\) which can only happen when \(f_{xx}\) and \(f_{yy}\) have the same sign.
  • When \(f_{xx}\) and \(f_{yy}\) have different signs, we are gauranteed to have \(f_{xx} f_{yy} - f_{xy}^2 < 0\), so it is a saddle point.

(Exercise)

  1. Find the critical point of \(f(x, y) = x^2 - xy + \frac{1}{2} y^2 + 3\).
  2. Determine whether the critical point is a local maximum, a local minimum, or a saddle point.
  3. Find the critical points of \(g(x, y) = 3y^2 - 2y^3 - 3x^2 + 6xy\).
  4. Determine whether the critical points are local maxima, local minima, or saddle points.

Solution.

  1. We have found in the previous exercise that $$\nabla f = \langle 2x - y, -x + y \rangle.$$ Setting the gradient to zero, we get $$2x - y = 0, \quad -x + y = 0.$$ Solving the system of equations, we get the critical point \((0, 0)\).
  2. We have found the second derivatives of \(f\) in the previous exercise, which are $$f_{xx} = 2, \quad f_{yy} = 1, \quad f_{xy} = -1.$$ Thus, we have $$f_{xx} f_{yy} - f_{xy}^2 = 2 \cdot 1 - (-1)^2 = 1 > 0,$$ and since \(f_{xx} > 0\), the critical point is a local minimum.
  3. We need to solve $$f_x = -6x + 6y = 0 \quad \text{and} \quad f_y = 6y - 6y^2 + 6x = 0.$$ From the first equation, we find \(x = y\), and substition for \(y\) into the second equation gives us $$- 6x^2 + 12 x = 0 \quad \text{or} \quad -6x (x - 2) = 0.$$ The solutions are \(x = 0\) and \(x = 2\), so the critical points are \((0, 0)\) and \((2, 2)\).

  4. We calculate the second derivateives $$f_{xx} = -6, \quad f_{yy} = 6 - 12y, \quad f_{xy} = 6.$$ The disciminant is $$f_{xx} f_{yy} - f_{xy}^2 = -36 + 72 y - 36 = 72(y - 1).$$

    • At the critical point \((0, 0)\), the discriminant is negative, so it is a saddle point.
    • At the critical point \((2, 2)\), the discriminant is positive, and since \(f_{xx} < 0\), it is a local maximum.
